# 🚀 TaskMaster API - Backend con .NET 10

Este proyecto es una **API RESTful** profesional para la gestión de tareas, construida con las últimas tecnologías de Microsoft. Demuestra el uso de una arquitectura limpia y la integración con bases de datos relacionales.

## 🛠️ Stack Tecnológico
* **Framework:** .NET 10 (ASP.NET Core)
* **Base de Datos:** SQL Server
* **ORM:** Entity Framework Core (Code First)
* **Documentación:** Scalar / OpenAPI
* **Control de Versiones:** Git

## ✨ Características Principales
* **CRUD Completo:** Creación, lectura, actualización y borrado de tareas.
* **Persistencia de Datos:** Conexión real a SQL Server para almacenamiento seguro.
* **Validación de Modelos:** Control de errores y tipos de datos (400 Bad Request).
* **Documentación Interactiva:** Interfaz visual para pruebas rápidas.


## 🚀 Cómo ejecutarlo
1. Clona el repositorio: `git clone https://github.com/humberto1997/TaskMasterAPI.git`
2. Configura tu `appsettings.json` con tu cadena de conexión local.
3. Ejecuta las migraciones: `dotnet ef database update`
4. ¡Dale a Play y listo!
